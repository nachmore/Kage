CALCULATOR APP STEERING:

When using the Calculator app:
1. The display shows the current value — look for elements with "Display is" in the name.
2. Button names are full words: "One", "Two", "Three", etc. (not "1", "2", "3").
3. Operators: "Plus", "Minus", "Multiply by", "Divide by", "Equals".
4. Use "Clear" to reset the calculator before starting a new calculation.
5. After pressing Equals, read the display to get the result.
