# Product Overview

Kiro Assistant is a cross-platform desktop AI assistant application with a floating window interface. It provides quick access to AI capabilities through a system tray application that can be activated via global shortcuts.

## Core Features

- Floating window interface for quick AI interactions
- System tray integration for background operation
- Custom command shortcuts for instant command execution without LLM processing
- Agent Communication Protocol (ACP) client for connecting to kiro-cli backend
- Cross-platform support (Windows, macOS, Linux)
- Settings management UI for configuration
- Application launcher integration (scan and launch installed apps)

## Key Components

- Floating window: Always-on-top, transparent, minimal UI for quick queries
- Settings window: Configuration interface for hotkeys, shortcuts, permissions, and appearance
- System tray: Background presence with quick access menu
- ACP client: Handles communication with the AI backend service
- Process manager: Manages spawned processes and child applications
- Application launcher: Discovers and launches system applications

## User Experience

Users interact with Kiro through a floating window that can be summoned via global hotkey. The interface supports both AI-powered queries and direct command shortcuts for frequently used tools and applications.
