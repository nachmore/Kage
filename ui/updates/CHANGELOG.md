## v0.2.0 — Initial Release

Welcome to Kage! Here's what's included in this first release:

### AI
- Floating window interface summoned via global hotkey
- Full chat window for longer conversations with session history
- Streaming responses with markdown rendering, code highlighting, and mermaid diagrams
- Image analysis — paste or drag images into the chat
- Selected text capture from any window as context for your prompts
- Tool permission management (allow, deny, ask per tool)

### Productivity
- Custom command shortcuts — run programs, open URLs, send prompts, or execute scripts
- AI-assisted script generation for shortcuts
- Inline math calculator that evaluates expressions as you type
- Application launcher with fuzzy search

### Personalization
- Auto-steering learns your preferences across sessions
- User steering document for explicit instructions
- Configurable AI model selection

### Desktop Integration
- System tray with background operation
- Dark, light, and system theme with configurable opacity
- Configurable window positioning (center, near mouse, remember last)
- Launch on startup
- Auto-update system with silent install

### Cross-Platform
- Windows, macOS, and Linux support
- Native OS integration for clipboard, hotkeys, and file management
