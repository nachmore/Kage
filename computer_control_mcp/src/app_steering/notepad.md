NOTEPAD APP STEERING:

When using Notepad:
1. Notepad opens directly to a blank document — no start screen to dismiss.
2. The main text area is an "edit" element — use set_value() or type_and_get_tree() to enter text.
3. To save: Use key_press("ctrl+s") which opens the Save dialog, or use File > Save As for a new file.
4. In the Save dialog, the filename field is an "edit" element — use set_value() to set the filename.
